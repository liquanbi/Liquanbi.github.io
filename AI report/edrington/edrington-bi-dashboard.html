BI 看板分析框架

数据驱动决策 · 品牌保护与市场洞察闭环

本框架旨在呈现分析思路与模块设计方向，最终 BI 交付物将以 Power BI 平台进行开发落地，实际呈现效果（包括图表样式、交互方式及布局细节）可能在此基础上有所调整与优化。

Framework Overview

数据分析故事线

从宏观市场表现 → 渠道合规监控 → 价格健康分析

FRAMEWORK OBJECTIVE
通过三大核心模块的联动分析，实现对爱丁顿品牌在电商渠道的全方位监控，及时识别市场机会与潜在风险，构建完整的品牌保护与市场洞察闭环。

Module 01

市场表现分析

MARKET PERFORMANCE ANALYSIS · 月度更新

CORE QUESTION
在当前市场结构下，每月市场表现如何？是增长还是下降？各渠道、各平台、各产品的表现如何？

分析维度

🏪

授权 vs 非授权渠道

官方渠道与第三方渠道销量对比，分析渠道健康度

🌐

平台维度

天猫、京东、淘宝各平台销售表现分析

📦

产品（SKU）维度

各单品销售表现追踪，识别明星产品

📍

大区 / 城市维度

地域销售分布分析，优化区域策略

分析指标

指标类别 具体指标 说明
销售指标 月销量、月销额 衡量市场规模
趋势指标 月同比（YoY）、月环比（MoM） 判断增长趋势
结构指标 各维度占比 分析市场结构
排行指标 店铺榜单、商品榜单 识别头部玩家

Module 02

非授权渠道监控

UNAUTHORIZED CHANNEL MONITORING · 月度更新

CORE QUESTION
EC 市场非授权渠道活跃情况如何？哪些店铺疑似销售假货？假货占比是否超过预警阈值？

分析维度

🏪

店铺维度

在售店铺数量、有销量店铺数量监控

🔗

链接维度

在售链接数量、产出销量链接统计

⚠️

疑似假货维度

假货链接数量、假货销量分析

📍

平台维度

分平台假货分布与风险等级评估

分析指标

监控对象 核心指标 预警阈值
店铺数量 在售店铺数、有销量店铺数 月环比 >20%
链接数量 在售链接数、产出销量链接数 月环比 >15%
疑似假货 假货链接数、假货销量 F/G Ratio >20%
折扣阈值 折扣 ≥65% 销量 占比 >10%

CORE QUESTION
产品价格表现如何？市场上主流折扣力度如何？正品与假货的价格差异有多大？

📦

Power BI 可视化建议

基于本框架，Power BI 可实现所有图表类型：
• 趋势分析：折线图、面积图、瀑布图
• 结构分析：环形图、树状图、桑基图
• 对比分析：条形图、矩阵表、卡片图
• 预警监控：仪表图、KPI 指标卡

支持数据自动刷新、交互式筛选，满足日常监控需求。
